Adaptive BI Framework 3

The Adaptive BI Framework aims to complete automate the Extract and the Load phase of an ETL project, be it for Business Intelligence or Data Integration. Extract and Load phase are just boring and always similar, so it would be nice to spare time spent to create such basic things in order to use it to do something clever.
